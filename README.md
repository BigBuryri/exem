## Модули 1–3 — Банкетам.Нет

Я сделал веб-портал для бронирования банкетных помещений: зал, ресторан, летняя и закрытая веранда.

**Стек:** React, Vite, Tailwind, Node.js, Express, PostgreSQL, Sequelize, JWT.

**Что реализовал:**
- регистрация и вход пользователей;
- подача заявок на бронирование с валидацией;
- личный кабинет и список своих заявок;
- админ-панель (смена статусов, сортировка);
- слайдер на главной, блок контактов, блок качества;
- ER-диаграмма и SQL-схема БД.

### Запуск

```bash
npm run install:all
npm run dev
```

- Сайт: http://localhost:3000  
- API: http://localhost:5000  
- Админ: **Admin26** / **Demo20**

## Модуль 4 — Код безопасности

Я сверстал прототип мобильного приложения **430×932 px**: 2 экрана, HTML + CSS, 4-колоночная сетка.  
Тексты и структура — по файлу `media/Текст.pdf` из задания.

**Файлы:**
- `index.html` — главная (поиск, статистика, кнопки «Войти» / «Зарегистрироваться»);
- `products.html` — продукты по категориям, у каждого кнопка «Подробнее».

Переход между экранами — через нижнее меню **Главная ↔ Продукты**.

Открыть локально: `Модуль 4 (вариативная часть)/index.html`
